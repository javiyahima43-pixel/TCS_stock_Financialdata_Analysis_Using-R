 📊 TCS Stock Analysis using R Programming

A professional stock market analytics dashboard developed using R to analyze Tata Consultancy Services stock performance through interactive charts and financial visualizations.

Key Features

* TCS Stock Price Trend Analysis
* Interactive Financial Charts 📈
* Volume and Market Performance Analysis
* Time-Series Visualization
* Data-Driven Insights Dashboard

Technologies Used

* R Programming

   Dataset

Stock market dataset used for analyzing TCS share performance and trends.

 🎯 Project Objective

To visualize and analyze TCS stock market performance using interactive R charts and financial analytics techniques.
